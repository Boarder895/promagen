06

Data Layer

platform-data.ts — Central SSOT

Merges platform-config.json + providers.json into AuthorityPlatform. Exports: getAuthorityPlatforms(), getAuthorityPlatformById(), getPlatformCounts(), getPlatformsByNegativeSupport(), getBqiHeadlineRange(), getLastUpdated(), getSSOTVersion().

comparison-pairs.ts — 8 Pairs

Editorial layer: slug, platform IDs, keyDifference, chooseWhen, workflowGuidance, FAQ. Build-time ID validation. Exports: getComparisonSlugs(), getComparisonPairBySlug(), getRelatedComparisons(), getReverseSlugs().

use-case-recommendations.ts — 4 Use Cases

Editorial layer: slug, directAnswer, 5 recommended platforms with reasoning, formatConsiderations, FAQ. Build-time ID validation. Exports: getUseCaseSlugs(), getUseCaseBySlug(), getRelatedUseCases().

presentation.ts — Shared Constants

Tier badge colours, negative support colours, card backgrounds, table cell padding, divider styles. Single source — zero duplication across pages.
